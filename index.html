Conways Game of Life.

As an html tag

Anyone with an interest in mathematics and computers has probably heard of Conways Game of Life and should recognize a running world when they see one. So I thought it would make a great WebComponent.

Tag attributes

rows
The game's number of rows, defaults to 50.
columns
The game's number of columns, defaults to 50.
timer
Set to "run" to start the iteration timer.
interval
The number of miliseconds between iterations, defaults to 1000.

Styling

From css the game-of-life element can be styled using the ::part() psudo element. Use ::part(cell) to select all of the game's cells, handy for setting transitions. Use ::part(living), ::part(deceasced) to respectivly select living or deceased cells.
